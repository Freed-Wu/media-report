media-report
=============

A backup of my homework.

内容：任选一视频，利用“会声会影”（ Corel Video Studio
）等视频处理软件进行处理。 对选定的视频进行剪接、转场、字幕、配 乐等处理；

实验报告：说明主要处理流程。

Dependent
---------

1.  A LaTex distribution. Such as [texlive].
2.  [njustthesis].

Install
-------

``` {.zsh}
git clone git@github.com:Freed-Wu/media-report
cd media-report
latexmk -pvc main.tex
```
